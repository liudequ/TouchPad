# 触摸板介绍视频流程（当前版本）

## 推荐时长
- 总时长：约 2 分 45 秒
- 画幅：横屏为主（可后期裁切竖屏短视频）

## 分段流程
1. 开场结果（0:00-0:20）
   - 直接演示成品操作：移动、滚动、点击。
   - 口播点出核心能力：USB/蓝牙、UI 可配置、跨平台、省电。

2. 项目定位（0:20-0:30）
   - 一句话说明目标：做一块可用、可配置、可跨设备的触摸板。

3. 元件清单（0:30-0:55）
   - 介绍主要元件与用途。
   - 可提示详细 BOM 见 `docs/bom.md`。

4. 3D 模型与结构（0:55-1:15）
   - 展示外壳模型与实物。
   - 说明当前模型较粗糙，主要用于尺寸与装配验证，空间仍有较大冗余。

5. 接线说明（1:15-1:25）
   - 只做一句话说明：详细接线见项目文档。
   - 可引用 `docs/fpc_pinout.md` 与 `docs/touchpad_usage.md`。

6. 功能介绍（1:25-2:05）
   - USB/蓝牙连接。
   - UI 可配置手势映射（左上角、右下角、三指等）。
   - 兼容 Android/Linux/Windows。
   - 省电策略（活动高响应、空闲低功耗）。

7. AI 参与（2:05-2:30）
   - 说明 AI 负责逻辑拆解和步骤建议。
   - 强调实作、调试和验证由本人完成。

8. 结尾（2:30-2:45）
   - 总结：可稳定使用、可按习惯配置、可跨设备使用。
   - 引导查看仓库文档与模型文件。

## 口播开场参考（30 秒）
“这是我做的一块触摸板。它支持 USB 和蓝牙连接，支持通过 UI 配置手势映射，比如左上角点击、右下角点击和三指手势；同时可连接 Android、Linux 和 Windows，并带有省电策略。接下来我会用最短时间讲清楚：需要哪些元件、3D 模型和结构、主要功能，以及我是怎么在 AI 辅助下把它做出来的。”
