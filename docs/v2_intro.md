# TouchPad V2 视频口播稿

这份文档是压缩到约 2 分钟的口播版，保留四段结构，适合直接录制。

## 推荐时长

- 总时长：约 2 分钟
- 适合形式：项目介绍、成品展示、视频口播

## 2 分钟口播稿

大家好，这次想介绍一下我做的 TouchPad V2。

它是一块基于 `nRF52840` 的触摸板，支持 `USB HID` 和 `BLE HID` 两种输出方式。基础操作上，它支持单指移动、双指滚动、轻触点击，同时也支持三指和四指手势，并且可以通过串口或者上位机 UI 来做参数和动作映射配置。

如果说前面的版本更像是在验证这个触摸板能不能跑起来，那 V2 更像是一个真正往成品方向推进的版本。因为这一版不只是把功能做通了，还补上了 PCB 底板、外壳，以及比较完整的文档和分享资料。

先说初始总结。V2 的核心目标不是做一个简单演示，而是尽量做成一个日常可用、可调、可复用，也方便别人理解和复现的版本。所以这一版最重要的提升，就是功能、结构和资料三件事开始连成闭环了。

接下来是 PCB 板介绍。V2 的 PCB 主要解决的是稳定连接、规整装配和方便复现这三个问题。当前方案基于 `nice!nano v2` 或者 `SuperMini nRF52840`，触摸板这一侧使用 `6P 1.0mm` FPC 接口，固件按 `I2C-HID` 方式读取，设备地址是 `0x2C`。这块板子的意义，不是电路多复杂，而是把原来飞线和临时接法，整理成了一块更稳定、更容易装配、也更适合分享的底板。

然后是外壳介绍。V2 现在已经配套了外壳模型，说明这个项目已经不只是电路和固件验证，而是开始认真处理整机形态。当然它现在还更偏工程验证版本，不是最终外观版，但它已经能把触摸板、PCB、主控和电池稳定装进同一套空间里，让这个项目从电子模块，变成一个真正可以拿在手里展示和使用的设备。

最后是 PCB 板分享。V2 现在已经整理出了比较完整的分享资料，包括 `Gerber` 生产文件、PCB 的 `STEP` 模型、外壳文件、BOM、FPC 引脚定义，以及固件烧录和使用说明。也就是说，分享出去的不只是“一张板子的截图”，而是一套别人可以看懂、可以组装、也可以继续修改的版本。

总结一下，TouchPad V2 的意义，不只是把触摸板跑起来，而是把 PCB、外壳、固件和文档慢慢整理成了一套完整方案。它还不是最终版，但已经从实验验证，走到了一个更接近成品、也更适合分享的阶段。
