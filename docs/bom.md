## 连接速查
- 触摸板 `VCC` -> 主控 `3V3`
- 触摸板 `GND` -> 主控 `GND`
- 触摸板 `SDA` -> 主控 `D9`
- 触摸板 `SCL` -> 主控 `D8`
- 触摸板 `INT` -> 主控 `D7`
- 触摸板 `EN` -> 主控 `D6`
- 微动复位（可选）：微动开关两端分别接主控 `RST` 与 `GND`，按下时短接触发复位。

## FPC 针脚速查（镜像后的当前定义）
- `Pin1` -> `VCC` -> 主控 `3V3`
- `Pin2` -> `SCL` -> 主控 `D8`
- `Pin3` -> `SDA` -> 主控 `D9`
- `Pin4` -> `INT` -> 主控 `D7`
- `Pin5` -> `ENABLE/EN` -> 主控 `D6`
- `Pin6` -> `GND` -> 主控 `GND`

注意：这里的 `Pin1~Pin6` 已按镜像后的实际顺序整理，不要再按旧丝印理解。

详见：`docs/fpc_pinout.md`。
